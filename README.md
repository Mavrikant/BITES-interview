# BİTES interview

M. Serdar Karaman (serdar@karaman.dev)

* Generic zone filling algorithm asked by BITES.
* Details of project can be found under ```doc``` folder

## Compile
* ```git submodule update --init --recursive``` get gtest library
* ```mkdir build```
* ```cd build```
* ```cmake ..```
* ```make all```

## Test
* ```cd /build/tst```
* ```./BITES_tst```

## Run
* ```cd /build/src```
* ```./BITES_run```
